# Survive Judge

VR/PC 1인칭 추리 게임입니다.

## 주의사항
이 프로젝트에는 크기 문제로 에셋이 포함되지 않았습니다.
게임 실행을 위해서는 별도의 구글 드라이브 파일을 압축 해제 후 사용해주세요.

VR용으로 제작하였지만, PC에서도 상호작용을 제외한 나머지는 플레이 할 수 있습니다.




### 실행
```bash
# 압축파일 다운로드 후 SurviveJudge.exe 파일 실행
```



## 조작법
```
- **Space**: 다이어로그 넘기기
- ***P***: 씬 이동

- *A, S, D, F, G*: 마인드아이템(박스) 파괴


- ***H***: 힌트 보기/안보기 토글
- 힌트 본 상태에서 *Space*: 힌트 얻기
- ***A***: 3D 문자 정답으로 진행하기

- **ESC**: 종료
```
